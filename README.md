### description of the project
The project is a simple E-Commerce system that models different types of products and
their interactions with customers. The project consists of the following classes:
- Product
- ElectronicProduct
- ClothingProduct
- BookProduct
- Customer
- Cart
- Order
- EcommerceSystem
